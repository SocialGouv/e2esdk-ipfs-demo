# e2esdk + IPFS demo

Demonstrate message and file encryption/decryption using e2esdk and browser IPFS for the storage.

## Usage

```
yarn && yarn dev
```

Login with a link like http://localhost:5173/#**mainKey**=[mainKey]&**userId**=[userId]
